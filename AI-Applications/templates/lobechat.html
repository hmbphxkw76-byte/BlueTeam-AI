AI 对话终端

基于 LobeChat 的现代化 AI 对话界面，提供媲美 ChatGPT 的交互体验。 支持 Markdown 渲染、代码高亮、对话分支和插件扩展。

🚀 启动 LobeChat

LobeChat 运行在独立端口，提供完整的聊天体验。点击下方按钮在新标签页中打开。

打开 LobeChat →

LobeChat URL http://localhost:3210
Access Code lobe-aiseclab

🔑 API 配置

API Proxy URL https://localhost:443/v1
API Key training-key

🤖 模型配置

默认模型 gpt-4o-mini
提供商 OpenAI 兼容

📋 使用说明

1

启动服务

确保通过 docker compose up -d 启动了完整服务栈。

2

打开 LobeChat

点击上方按钮访问 localhost:3210，输入 Access Code。

3

开始对话

LobeChat 已预配置连接本地 AI 服务，直接开始对话即可。

⚠️ 注意： LobeChat 需要单独通过 Docker Compose 启动。如果尚未配置，请在 docker-compose.yml 中添加 LobeChat 服务。 如果 LobeChat 未启动，仍可使用本平台的 内置聊天界面。
