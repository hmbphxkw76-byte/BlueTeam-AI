# USB-C 高速数据线

## 产品规格
- 长度: 1米
- 接口: USB-C to USB-C
- 协议: USB 3.2 Gen2
- 充电: 100W PD 快充
- 传输: 10Gbps

## 使用说明
1. 连接设备的 USB-C 接口
2. 确保接口完全插入
3. 支持热插拔

## 兼容设备
- 所有 USB-C 接口设备
- MacBook Pro/Air (2016+)
- iPad Pro
- Android 手机
- 支持 PD 充电的设备

## 安全提示
- 请勿在潮湿环境中使用
- 避免过度弯折接口处
- 使用原装充电器
